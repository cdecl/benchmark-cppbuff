# benchmark-cppbuff


### Submodule Init 

```
git clone https://github.com/cdecl/benchmark-cppbuff.git
cd benchmark-cppbuff

git submodule init
git submodule update
```


### Build 

```
# Make Build Directory
$ mkdir build 
$ cd build

# cmake 
$ SET CMAKE_CXX_COMPILER=g++ 
$ cmake -G "MinGW Makefiles" ..

# make build
$ make 
```

### Run
```
$ main
```
